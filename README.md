# planeshift

Project Builder CLI

## Install

```
$ npm install -g planeshift
```


## Usage
```
$ planeshift [app] [path]
```

**example**

```
$ planeshift 70-10/node-boilerplate exapmle-project
```

### [app]

Project repository.  
`username/repository`

### [path]

Create project path.  
`default: . (current directory)`
